How to use Latch

  • Configuration
  • Pairing / Unpair
  • Devices
  • Administration

Configuration

In the configuration screen we have to add the parameters that gives us the application of Latch when creating it.

The parameters Application ID and Secret Key are generated automatically by the Latch Application.

The reload time parameter is the time it takes for the system to re-check all wireless devices connected. The minimum time is 15 seconds although 20 seconds is recommended.

Pair

In the pairing screen we can add a device (mobile phone, tablet...) to control the system and block/unblock access to the network for the wireless devices that are trying to connect

The Username parameter is used to recognise the control device. You can put any alias you want.

The Pair code parameter is generated by the control device to pair with the plugin and start to control this system.

Unpair

In the unpairing screen you can unpair the control device to add other

You only need to use the button with the alias name to unpair it

Devices

In the devices list you can add a Permanent Mac and manage the connected devices

You can add a Permanent Mac using a alias to recognise it and the mac of the device. If you add a mac as permanent you wont manage it with the control device and will connect automatically without problems, but this is less secure than control all connected devices with the control device

You have a list of devices connected to see some interesant info and the posibility to edit the devices

Editing Devices

In the Device editing screen you can put a alias to the device to recognise it and you can establish it as permanent mac

The alias parameter is only a name than you will use to recognise the device. For instance: "My Tablet"

You have the posibility of add the device as permanent. If you put the mac as permanent this will disapear from your control device and will connect automaticaly and without control. Remember that this is less secure than control all devices with the control device

if you want to delete a permanent mac you only need to unmark this option in the device editing screen and the system will delete it and start to control manually

Managing the service

Full integration

You can easily manage the plugin in the System > Startup screen.

You will see that the plugin is fully integrated in OpenWrt and you can start/restart/stop the plugin and set or unset autostart in boot easily

Uninstalling the plugin

Easy uninstalling

Thanks to the full integration in OpenWRT you can easy uninstall the plugin going to System > Software and clicking in "Remove"
